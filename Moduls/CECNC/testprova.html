    Qüestionari de Desenvolupament Web

    Pregunta 1: Quins dels següents són llenguatges utilitzats en el desenvolupament web del costat del client? (Selecciona totes les opcions correctes)

    Pregunta 2: Quin protocol s'utilitza per transferir pàgines web de manera segura? (Selecciona només una opció)

    Puntuació Final:

    Justificació Pregunta 1:

    Les respostes correctes són JavaScript, HTML i CSS:

    • JavaScript: Llenguatge de programació que s'executa al navegador per afegir interactivitat.
    • HTML: Llenguatge de marcat per estructurar el contingut de les pàgines web.
    • CSS: Llenguatge d'estils per definir l'aparença visual dels elements HTML.

    Respostes incorrectes:

    • PHP: És un llenguatge del costat del servidor, no del client.
    • Python: Encara que pot usar-se en web, s'executa principalment al servidor, no al client.

    Justificació Pregunta 2:

    La resposta correcta és HTTPS:

    • HTTPS: Extensió segura d'HTTP que utilitza TLS/SSL per encriptar les comunicacions entre el client i el servidor.

    Respostes incorrectes:

    • HTTP: Protocol per transferir pàgines web, però sense encriptació.
    • FTP: Protocol per transferir fitxers, no específicament per pàgines web.
    • SMTP: Protocol per enviar correus electrònics, no per pàgines web.
