    Vinculació de Tasques del Projecte Intermodular DAW amb Resultats d'Aprenentatge dels Mòduls Professionals¶

    He creat una vinculació completa de les 95 tasques del Projecte Intermodular (MP12) amb els resultats d'aprenentatge específics dels mòduls professionals del cicle DAW.12345678

    Estructura de la Vinculació¶

    La vinculació considera els següents mòduls professionals del cicle DAW:

    • MP01: Sistemes Informàtics (198h) - 7 RA23
    • MP02: Bases de Dades (198h) - 7 RA682
    • MP03: Programació (198h) - 6 RA692
    • MP04: Llenguatges de Marques i Sistemes de Gestió d'Informació (99h) - 3 RA370
    • MP05: Entorns de Desenvolupament (66h) - 6 RA7172
    • MP06: Desenvolupament Web en Entorn Client (165h) - 7 RA733
    • MP07: Desenvolupament Web en Entorn Servidor (165h) - 9 RA43
    • MP08: Desplegament d'Aplicacions Web (66h) - 6 RA578
    • MP09: Disseny d'Interfícies Web (99h) - 4 RA743
    • MP10: Formació i Orientació Laboral (99h) - 4 RA2
    • MP12: Projecte de Desenvolupament d'Aplicacions Web (99h) - 5 RA65

    Exemples de Vinculacions per Blocs¶

    Bloc 1 - Identificació i Anàlisi de Necessitats¶

    Tasca RA Vinculats Justificació
    1.1 Investigació del mercat tecnològic MP05.RA1, MP05.RA2, MP10.RA4 Reconeix elements i eines desenvolupament programari, avalua entorns IDE i desenvolupa competències emprenedores7172
    1.5 Anàlisi de viabilitat tècnica MP01.RA1, MP01.RA5, MP08.RA1 Avalua sistemes informàtics, interconnecta sistemes en xarxa i implanta arquitectures web28
    1.8 Selecció de tecnologies MP05.RA1, MP06.RA1, MP07.RA1, MP08.RA1 Selecciona arquitectures i tecnologies per client, servidor i desplegament7148

    Bloc 2 - Disseny i Planificació¶

    Tasca RA Vinculats Justificació
    2.3 Diagrama de classes MP05.RA5, MP03.RA4 Genera diagrames de classes UML i aplica programació orientada a objectes7169
    2.9 Disseny del model de dades MP02.RA2, MP02.RA6 Crea bases de dades definint estructura segons model relacional i dissenya diagrames ER682
    2.12 Wireframes i mockups MP09.RA1, MP09.RA2 Planifica interfície web valorant especificacions de disseny i crea interfícies homogènies7475

    Bloc 3 - Desenvolupament i Implementació¶

    Tasca RA Vinculats Justificació
    3.3 APIs RESTful MP07.RA7, MP07.RA9 Desenvolupa serveis web i aplicacions híbrides amb tecnologies servidor4
    3.9 Interacció UI MP06.RA5, MP06.RA6, MP09.RA2 Desenvolupa aplicacions interactives, gestiona esdeveniments i manipula el DOM373
    3.14 Consultes optimitzades MP02.RA3, MP02.RA5, MP07.RA6 Consulta informació emmagatzemada, desenvolupa procediments i accedeix eficientment a dades684

    Bloc 4 - Documentació i Presentació¶

    Tasca RA Vinculats Justificació
    4.1 Manual tècnic - Arquitectura MP08.RA6, MP05.RA5, MP12.RA5 Elabora documentació aplicació, genera diagrames tècnics i avalua projecte8716
    4.12 API docs Swagger MP07.RA7, MP08.RA6 Desenvolupa serveis web amb documentació autogenerada i eines col·laboratives48

    Bloc 5 - Avaluació i Millora Contínua¶

    Tasca RA Vinculats Justificació
    5.1 Testing unitari backend MP05.RA3, MP07.RA4 Verifica funcionament dissenyant proves i desenvolupa aplicacions servidor714
    5.9 Auditoria accessibilitat WCAG MP09.RA4, MP06.RA1 Aplica criteris d'accessibilitat i usabilitat segons estàndards743
    5.18 Desplegament aplicació MP08.RA3, MP08.RA4, MP08.RA5 Administra servidors transferència, implanta aplicacions i verifica execució578
    5.24 Avaluació crítica global MP12.RA4, MP12.RA5 Prepara i efectua presentació del projecte, reflexiona sobre competències desenvolupades6
